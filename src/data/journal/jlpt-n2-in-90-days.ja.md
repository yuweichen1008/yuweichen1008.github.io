# 90日でJLPT N2：僕の勉強スタック

2年前、JLPT N3に落ちた。勉強していなかったわけじゃない——していた。失敗したのは、間違った順序で間違ったものを勉強して、聴解でパニックになったから。練習していた何よりも速かった。

今回は、運任せにしない。

## なぜN2で、なぜ今か

N2は、日本で仕事するのに実際にドアを開けるベンチマークだ。N3は「日本語を勉強した」と言える。N2は「機能できる」と言える。

ターゲットは2026年7月の試験。4月1日からだとおよそ90日。きつめだけど、スケジュールが保てれば可能だ。

## スタック

### 1. Anki——語彙の骨格

JLPT N2 Core 6000デッキを使っている。1日30枚の新カード、同じセッションで期限切れのものを復習。

守っているルール：3秒以内に思い出せないカードはミスとしてマークする。部分点なし。躊躇は、まだ本当には記憶に入っていないということ。

1日30枚 × 90日 = 2,700枚の新カード。デッキの間隔反復で、7月までにテスト対象の語彙の約90%を保持できるはず。

### 2. げんきII + N2文法参考書

シンガポールでげんきIを終えた。げんきIIはN4/N3レベルの文法をカバーする。終わった後、Try! N2文法テキストに切り替えた——文法ポイントごとに整理されていて、文脈豊かな例文がある。

方法：1日1つの新しい文法ポイント。例文を3つ書く。下の日記書きで使う。

### 3. 日本語日記

毎日、その日の出来事を日本語で3〜5文書く。複雑じゃなくていい——食べたもの、行った場所、感じたことを記録するだけ。これが日本語を*産出する*習慣を作る。消費するだけでなく。

英語を学んでいるランゲージエクスチェンジパートナーとこれを共有している。週1回お互いに訂正し合う。

### 4. 聴解——N3で負けた理由

ここが前回の失敗だった。N2の聴解は速く、問題は繊細——推論をテストする、単純な理解ではなく。

僕のアプローチ：
- **NHK Web Easy** 毎日——簡略化された日本語ニュース。読んで、また声に出して読む
- **シャドーイング**：シャドーイング日本語の本を使って、1日10分
- **JLPT聴解練習テスト**：6週目以降、週1回フルセクション

### 5. 読解——新聞とネイティブコンテンツ

4週目から、通常難度のNHK記事を毎日1本読み始めた。目的は完全な理解ではない——漢字が密な文章をペースを保って読むことに慣れること。

## スケジュール

| 週 | フォーカス |
|------|-------|
| 1〜3  | 語彙 + げんきII復習 + 聴解基礎 |
| 4〜6  | N2文法 + 読解入門 + シャドーイング |
| 7〜10 | 模擬テスト全般 + 集中聴解 + 語彙復習 |
| 11〜12 | 制限時間内模擬試験 |

## 正直な中間チェック

これを書いている時点で、90日の約半分を過ぎた。語彙は来ている。文法はほぼ揃っている。聴解はまだ弱点——前よりはわかるが、N2のスピードにたまについていけなくなる。

日記書きが予想外に一番役立っている。産出することで、「知っていると思っていること」と「実際に知っていること」の差に向き合わざるを得なくなる。

7月が答えを出す。
