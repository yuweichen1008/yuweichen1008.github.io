# 90天衝JLPT N2：我的學習組合

兩年前，我的JLPT N3沒過。不是因為沒有準備——我有準備。失敗是因為我用錯誤的順序學了錯誤的東西，然後在聽力部分，當速度比我練習過的任何東西都快時，我慌了。

這一次，我不打算讓它留到運氣。

## 為什麼是N2，為什麼是現在

N2是在日本職場實際打開大門的基準。N3說的是「我學過日文」。N2說的是「我可以正常運作」。

我的目標是二○二六年七月的考試，從四月一日算起大約給了我九十天。很緊，但只要時程撐住是可行的。

## 學習組合

### 1. Anki——詞彙骨架

我使用JLPT N2 Core 6000牌組。每天30張新卡，同一個時段複習到期的卡片。

我遵守的規則：如果三秒內無法想起一張卡，就標記為錯誤。沒有部分得分。猶豫意味著它還不在記憶裡。

每天30張 × 90天 = 2700張新卡。透過預載牌組的間隔重複，到七月時我應該對測驗詞彙有約90%的保留率。

### 2. Genki II + N2文法參考書

我在新加坡完成了Genki I。Genki II涵蓋N4/N3左右的文法。完成之後，我換成了Try! N2文法教科書——它按文法點組織，有豐富情境的例句。

方法：每天一個新文法點。寫三個例句。在下面的日記寫作中使用它們。

### 3. 日語日記

每天，我用日文寫3到5個句子記錄我的一天。不要求複雜——只是記錄我吃了什麼、去了哪裡、感受如何。這建立了*產出*日文的習慣，而不只是消費。

我與一位學英文的語言交換夥伴分享這些。我們每週互相糾正一次。

### 4. 聽力——N3的殺手

這是我上次失敗的地方。N2聽力速度快，問題很微妙——它們測試推論，不只是理解。

我的方法：
- **NHK Web Easy** 每天——簡化的日文新聞，讀完後再大聲朗讀一遍
- **影子跟讀**：使用《Shadowing Japanese》，每天十分鐘
- **JLPT聽力模擬題**：從第六週起，每週一個完整的聽力部分

### 5. 閱讀——報紙和原生內容

從第四週起，我開始每天閱讀一篇正常難度的NHK文章。目標不是完整理解——而是習慣以正常速度閱讀充滿漢字的文本。

## 時程

| 週數 | 重點 |
|------|------|
| 1–3  | 詞彙 + Genki II複習 + 聽力基礎 |
| 4–6  | N2文法 + 閱讀入門 + 影子跟讀 |
| 7–10 | 完整模擬題 + 密集聽力 + 詞彙複習 |
| 11–12 | 限時模擬考 |

## 誠實的中途檢視

我在寫這篇的時候大約過了一半的九十天。詞彙在進步中。文法大部分到位了。聽力依然是我的弱項——我理解的比以前多了，但N2的速度偶爾還是會讓我跟不上。

日記寫作是意外最有用的部分。產出輸出強迫你面對你實際上不會的東西，與你以為自己會的東西的差距。

七月會告訴我們答案。
