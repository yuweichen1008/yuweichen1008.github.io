Introduction

This is the introduction of GPON. Contains Archtecture, activation processes and comparison to EPON, NG-PON and more.

Network Architecture

    • Reconciliation Sublayer (RS)
    • Physical Media Attachment (PMA)
    • Physical Medium Dependent
    • Medium Dependent Interface / GMII

GPON Activation Processes

  • Initial State (O1)

    In this state, the ONU switches on, waits for the DS signal.
    Once the DS transmission is received, the LoS and LoF are cleared, the DS transmission is synchronized, and the ONU moves to the O2 State.

  • Standby State (O2)

    ONU waits for global network parameters.
    Once the US overhead message is received, the ONU sets up the assigned parameters and moves to the O3 state

  • Serial Number State (O3)

    1. ONU waits for unique ONU-ID sent by OLT’s AssignONU-ID message. Once the number is assigned, the ONU moves to the next state.
    2. The OLT can also send an Extended Burst Length message to all connected ONUs and hand over the extended overhead parameters.
    • OLT will create quiet window with a duration of 250 us by sneding a frame with an empty bandwidth map (BWmap) field

  • Ranging State (O4)

    • In order to make ONUs appear to be equal distance from OLT. The Equcal Delay(EqD) is measured and once ONU receives the Ranging-Time message, it moves to O5 State
    • During this particular state, a quiet window with a duration of 202 us is created.

  • Operation State (O5)

    • ONU can send US data, PLOAM messages and arrive separately in its correct location within the US GTC frame
    • Once the network is ranged, and all the ONUs are working with their correct
      equalization delay, all upstream bursts will be synchronized together between all the ONUs.

  • POPUP State (O6)

    ONU enters this state when any of the LoS or LoF alarms is detected.
    ONU stop ending data. Then, attempts to retrieve the optical signal, recover the synchronization of the GTC frame and remove the LoS / LoF alarm.

  • Emergency Stop State (O7)

    An ONU that receives a Disable Serial Number with the “deactivate” option goes to this state and shuts off the laser.
    The activation is accomplished by sending the Disable Serial Number message with the “enable” option. And return to O2.
